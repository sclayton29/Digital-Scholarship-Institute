# Tools

This file collects useful tools discussed at the Institute, whether formally in workshops or informally at the snack table.

- [Color blindness simulator](http://www.color-blindness.com/coblis-color-blindness-simulator/)
- [ColorBrewer2](http://colorbrewer2.org/)

## Data visualization
- [DataWrapper](https://www.datawrapper.de/)
- [Rawgraphs](https://rawgraphs.io/)
- [Tableau Public](https://public.tableau.com/s/)

- [Open Data Kit for mobile](https://opendatakit.org/)

## Dataset creation
- [JSTOR Data For Research](https://www.jstor.org/dfr/)
- [HathiTrust Worksets builder](https://analytics.hathitrust.org/staticworksets)
